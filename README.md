<h1 align="center">Skill Sharing Platform</h1>

<p align="center">
  A PHP and MySQL web application for sharing skills, with teacher and user roles,
  authentication, validation, and secure password hashing.
</p>

<hr>

<h2>📌 Overview</h2>
<p>
  This project was developed as part of a university course. It is a web-based skill-sharing
  platform where users can create accounts and interact based on their roles.
</p>

<p>
  The application supports two types of accounts:
</p>

<ul>
  <li><strong>Teacher accounts</strong> – can share and manage skills</li>
  <li><strong>User accounts</strong> – can browse and interact with available skills</li>
</ul>

<p>
  The main goal of this project was to practice backend development with PHP,
  database integration with MySQL, and secure user authentication.
</p>

<hr>

<h2>🚀 Features</h2>
<ul>
  <li>User registration and login system</li>
  <li>Role-based access control (Teacher / User)</li>
  <li>Secure password hashing</li>
  <li>Input validation and error handling</li>
  <li>Session-based authentication</li>
  <li>MySQL database integration</li>
  <li>CRUD operations for managing skills</li>
</ul>

<hr>
